Lesson 3: CSS Basics

CSS (Cascading Style Sheets) is the code that styles web content. Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red:

p {
color: red;
}

The Anatomy of a CSS Rule Set

The structure provided in the image above is called a ruleset. The structure contains several individual parts as you may have noticed:

Selector:This is the HTML element name at the start of the ruleset which defines the elements to be styled.

Declaration:This is the HTML element name at the start of the ruleset which defines the elements to be styled.

Properties:This is the HTML element name at the start of the ruleset which defines the elements to be styled.

Property value:This is the HTML element name at the start of the ruleset which defines the elements to be styled.

  • Apart from the selector, each rule set must be wrapped in curly braces. ({})
  • Within each declaration, you must use a colon (:) to separate the property from its value or values.
  • Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
