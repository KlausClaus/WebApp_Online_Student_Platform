Lesson 1: Introduction to Web Development

In this course, you will learn all about becoming a web developer and what tasks a web developer performs on a day-to-day basis. You will learn how to build a simple website, as well as the basics of HTML, JavaScript and CSS. And, of course, we also have some code examples for you to try out, which you can use as templates for your own website. We’ll introduce you to the fundamental building blocks of the web and make sure you leave this course being able to develop your very own website!

So, what does it take to become a web developer? Essentially, there are only three things that you need: HTML, CSS and JavaScript, which you will learn how to use in this course. Together, these three pillars make every website work, defining the content to be displayed (HTML), telling a browser how to display that content (CSS), and making the content interactive (JavaScript), respectively.

Tools that will be needed for this course:

  • A text editor, which you will use to write code
  • A web browser, which allows you to see what we’re building
